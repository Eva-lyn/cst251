Fancy CSS 1

Challenge

The goal of this lab was to work with my partner, create a lab 15 index.html and create a site that used fancy CSS. I had to make elements that used box-shadow, text-shadow, gradient, filters, rounded corners, object-fit and transform.

For this lab I;

  • redesigned my lab site,
  • utilized rounded corners, text and box shadows to make neon lights,
  • used gradients for my content blocks,
  • used object fit and transform for my images,
  • created an extra tacky site using filters and everything in-between.

    • Problems

    I first started working on my tacky site but I ended up really liking it. I renamed my tacky site to be my index.html. I got so excited I even made graphics for my site. I themed this lab around music.

    I had display issues with my images for a while. The transform would make them move and overlap and glitch out for the longest time.

    I used filter blur for my extra tacky site because I couldnt find a way to incoorporate it into my index.html.
