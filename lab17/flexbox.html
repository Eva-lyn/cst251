CSS Layout

Challenges

For the flexbox page we were told to use flexbox elements in css so that our page functioned with flexboxes. I do like the way the flexbox element looks. I didn't have to worry about aligning the main block with the aside block because flexbox took care of that for me. It also allowed me not to worry about filling space. I'm excited to apply flexbox to my personal lab layout because it would resolve my block size and aligning issue.

Problems

  • The main issue I had was with getting my flexboxes to wrap the way I wanted them to.
  • There was also an issue with the size of my blocks. Even though they were set to 100% of the page, they exceeded the width of my screen size once I added borders and padding. I've never had this problem before so I really don't know what it is.
  • I struggled with the effect my blocks take when they wrap when shrunken. I wanted them to expand to 100% once they wrapped under eachother but I wasn't able to figure out how to do that.
