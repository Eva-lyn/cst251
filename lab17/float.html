CSS Layout

Challenges

For this page's layout we had to use only float as a way to make our page function. For this page I worked with my partner, however my page might be different because we werent able to finish our meeting because css wasnt working for a while.

Problems

I'm pretty familiar with float now so this page wasnt too hard. You cant really do much with float except for positioning how a block is orientated. The best way to use float for layout is in combination with positioning. For this lab I didn't use positioning to fix up my layout.

Results

I decided to keep my layout as a basig wireframe to show how the pure layout looks. Ive noticed that using float for layout is very limiting but it still gets the job done, especially for simple block layouts. I tend to use float a lot when I have blocks inside of blocks. Using center aligning really fixes up how it all looks. The best example of how I've been using float is in my nav bar.
